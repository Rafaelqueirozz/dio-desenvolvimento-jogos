Fundamentos do CSS

Introdução ao CSS

Resumindo em apenas uma frase:
Determina como deve ser o layout de uma página e como os elmentos do nosso site devem ser

  • O CSS é uma linguagem de estilos
  • O CSS não é uma linguagem de programação
  • O CSS não é uma linguagem de marcação

O que pode ser criado com CSS?

  • Layouts e estilização de páginas web
  • Animações
  • Formas geométricas e desenhos
  • Filtros
  • Contadores

Formas de declaração do CSS

O CSS precisa ser declarado através de propriedades e valores, acessando os seletores do HTML (ex: div, h1, classes, ids, pseudo classes, etc.)

Propriedade: é uma característica de um elemento do nosso HTML (ex: cor de fundo, largura, altura, etc)

Valor: define o resultado de uma propriedade e como o navegador deve exibir o estilo daquele elemento

propriedade: valor; exemplo: 


            seletor {
                background: red;
                color: white;
            }
3 Maneiras de incluir o CSS no seu HTML:
  • Inline - Dentro da prórpria da tag no html é inserido o estilo
  • Interno - Na área head do html é feito um estilo para os seletores
  • Externo - Um arquivo externo do tipo css é carregado
  • Ordem de prioriorades: inline > interno > externo
