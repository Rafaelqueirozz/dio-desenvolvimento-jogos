Textos - Word-wrap / Word-break

Propriedade word-wrap

Define, caso uma palavra seja muito grande para caber em uma linha, se ela será quebrada em qualquer caractere para que uma parte dela fique na liha e o resto na próxima. Ou se toda palavra deve ficar na mesma linha e ultrapassar os limites que definimos para o elemento.

Exemplos:

word-wrap: normal;

Trilha de CSS TrilhaDeCSSTudoJunto Trilha de CSS

word-wrap: break-word;

Propriedade word-break

Define quando o texto deve ter uma quebra de linha

Exemplos:

word-break: normal;

word-break: break-all;
